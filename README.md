# WSU Icons
An officially branded WSU icon set.

## Build Icon Font
1. Download repo locally
2. `npm i`
3. `npm run build`

## How to Use
- In CSS 
```css
content: "\F103"; 
font-family: "wsu-icons";
```
- In HTML 
```html
<span class="wsu-icon wsu-i-alarm"></span>
```
